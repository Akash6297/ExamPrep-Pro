# 🎓 ExamPrep Pro v5.0

**ExamPrep Pro** is a powerful, locally-first web application designed for students and exam aspirants of all kinds (MTS, GDS, Postman, SSC, etc.). It transforms your study materials (PDFs and Images) into interactive, timed, and scored mock tests.

## 🚀 Key Features

- **Multi-Modal Extraction**: 
    - **PDF Parsing**: Instantly extract questions and options from study material.
    - **AI OCR (Local)**: Take photos or screenshots of text-books/papers and convert them into tests using built-in Tesseract.js.
- **Test Arena**:
    - Real-time **Count-down Timer**.
    - **Negative Marking** support (-0.25, -0.33, etc.).
    - **Question Shuffling** for fresh practice every time.
    - **Navigation Grid** to track Answered/Skipped questions.
- **Smart Result Review**: 
    - Review every single mistake after a test.
    - **Re-take Exam**: Re-try any past test immediately to perfect your score.
- **100% Privacy**: No data ever leaves your computer. Everything is stored in your browser's **IndexedDB**.
- **Super Responsive**: Works perfectly on Mobile, Tablet, and Desktop.

## 🛠️ Developed With
- **Core**: HTML5, CSS3 (Glassmorphism), ES6 JavaScript
- **Libraries**: `pdf.js` (for PDF), `Tesseract.js` (for OCR), `Lucide` (for Icons).

## 📖 How to Run
1. Clone or Download this repository.
2. Open `index.html` in any modern web browser (Chrome, Edge, Safari).
3. (Optional) For the best experience with PDF processing, use a local server (like VS Code's **Live Server** extension).
